node-elementtree
====================

node-elementtree is a [Node.js](http://nodejs.org) XML parser and serializer based upon the [Python ElementTree](http://effbot.org/zone/element-index.htm) module.

Status
====================
Incomplete. Basics work. Major work ongoing.
